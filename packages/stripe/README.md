# Reign Auth Stripe Plugin

Stripe plugin for [Reign Auth](https://www.auth.reign-labs.com) — integrate Stripe billing with your authentication system.

## Installation

```bash
npm install @reign-auth/stripe
```

## Documentation

For full documentation, visit [auth.reign-labs.com/docs/plugins/stripe](https://www.auth.reign-labs.com/docs/plugins/stripe).
