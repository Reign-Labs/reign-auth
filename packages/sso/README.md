# Reign Auth SSO Plugin

Single Sign-On plugin for [Reign Auth](https://www.auth.reign-labs.com) — add SAML and OIDC enterprise SSO to your application.

## Installation

```bash
npm install @reign-auth/sso
```

## Documentation

For full documentation, visit [auth.reign-labs.com/docs/plugins/sso](https://www.auth.reign-labs.com/docs/plugins/sso).
