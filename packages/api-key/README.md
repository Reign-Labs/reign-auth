# Reign Auth API Key Plugin

API Key plugin for [Reign Auth](https://www.auth.reign-labs.com) — manage API keys for your users.

## Installation

```bash
npm install @reign-auth/api-key
```

## Documentation

For full documentation, visit [auth.reign-labs.com/docs/plugins/api-key](https://www.auth.reign-labs.com/docs/plugins/api-key).
