# Reign Auth i18n Plugin

Internationalization plugin for [Reign Auth](https://www.auth.reign-labs.com) — translate error messages and UI strings.

## Installation

```bash
npm install @reign-auth/i18n
```

## Documentation

For full documentation, visit [auth.reign-labs.com/docs/plugins/i18n](https://www.auth.reign-labs.com/docs/plugins/i18n).
