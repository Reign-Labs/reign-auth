# Reign Auth Prisma Adapter

Prisma adapter for [Reign Auth](https://www.auth.reign-labs.com).

## Installation

```bash
npm install @reign-auth/prisma-adapter
```

## Documentation

For full documentation, visit [auth.reign-labs.com/docs/adapters/prisma](https://www.auth.reign-labs.com/docs/adapters/prisma).
