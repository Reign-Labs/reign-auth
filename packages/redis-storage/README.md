# Reign Auth Redis Storage

Redis storage for [Reign Auth](https://www.auth.reign-labs.com) secondary storage — use Redis for session caching and rate limiting.

## Installation

```bash
npm install @reign-auth/redis-storage
```

## Documentation

For full documentation, visit [auth.reign-labs.com/docs/storage](https://www.auth.reign-labs.com/docs/storage).
