# Reign Auth Drizzle Adapter

Drizzle ORM adapter for [Reign Auth](https://www.auth.reign-labs.com).

## Installation

```bash
npm install @reign-auth/drizzle-adapter
```

## Documentation

For full documentation, visit [auth.reign-labs.com/docs/adapters/drizzle](https://www.auth.reign-labs.com/docs/adapters/drizzle).
