# Reign Auth OAuth Provider Plugin

OAuth provider plugin for [Reign Auth](https://www.auth.reign-labs.com) — turn your application into an OAuth 2.0 provider.

## Installation

```bash
npm install @reign-auth/oauth-provider
```

## Documentation

For full documentation, visit [auth.reign-labs.com/docs/plugins/oauth-provider](https://www.auth.reign-labs.com/docs/plugins/oauth-provider).
