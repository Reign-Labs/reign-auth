# Reign Auth SCIM Plugin

SCIM (System for Cross-domain Identity Management) plugin for [Reign Auth](https://www.auth.reign-labs.com) — enable enterprise user provisioning.

## Installation

```bash
npm install @reign-auth/scim
```

## Documentation

For full documentation, visit [auth.reign-labs.com/docs/plugins/scim](https://www.auth.reign-labs.com/docs/plugins/scim).
