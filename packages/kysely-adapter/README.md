# Reign Auth Kysely Adapter

Kysely adapter for [Reign Auth](https://www.auth.reign-labs.com).

## Installation

```bash
npm install @reign-auth/kysely-adapter
```

## Documentation

For full documentation, visit [auth.reign-labs.com/docs/adapters/kysely](https://www.auth.reign-labs.com/docs/adapters/kysely).
