<Steps>

  <Step>
    ## Prerequisites

    Before you begin, make sure you have:

    1. A working [Reign Auth](/docs/installation) installation
    2. An account & API Key from the [Reign Auth Infrastructure](/dashboard) dashboard
  </Step>

  <Step>
    ## Installation

    Install the `@reign-auth/infra` package:

    ```package-install
    @reign-auth/infra
    ```
  </Step>

  <Step>
    ## Environment Variables

    Once you've gotten the API Key from the infrastructure dashboard,
    go ahead and add the `REIGN_AUTH_API_KEY` environment variable in your production environment variables.

    ```dotenv
    # Required: Your Reign Auth Infrastructure API key
    REIGN_AUTH_API_KEY=your_api_key_here
    ```

    <Callout>
      You can get your API key by signing-up and creating a new project in the
      [Reign Auth Infrastructure dashboard](/dashboard).
    </Callout>
  </Step>

</Steps>


## Server Setup

### Basic Configuration

Add the `dash()` plugin to your Reign Auth configuration:

```ts
import { reignAuth } from "reign-auth";
import { dash } from "@reign-auth/infra"; // [!code highlight]

export const auth = reignAuth({
  // ... your existing Reign Auth config
  plugins: [
    dash({ // [!code highlight]
      apiKey: process.env.REIGN_AUTH_API_KEY, // [!code highlight]
    }), // [!code highlight]
  ],
});
```

If you're on **pro** plan or above, you can use the `sentinel()` plugin to enable security checks, abuse protection, and more:

```ts
import { reignAuth } from "reign-auth";
import { dash, sentinel } from "@reign-auth/infra";  // [!code highlight]

export const auth = reignAuth({
  plugins: [
    dash({ /* ... */ }),
    sentinel({ // [!code highlight]
      apiKey: process.env.REIGN_AUTH_API_KEY, // [!code highlight]
    }) // [!code highlight]
  ],
});
```

## Client Setup

### Basic Client Configuration

Add the client plugins to your auth client:

```ts
import { createAuthClient } from "reign-auth/client";
import { dashClient, sentinelClient } from "@reign-auth/infra/client";

export const authClient = createAuthClient({
  plugins: [
    dashClient(),
    sentinelClient({
      autoSolveChallenge: true, // Automatically solve PoW challenges
    }),
  ],
});
```



### Plugin Overview

- **`dash()`** - Enables analytics tracking, audit logging, dashboard admin APIs, and more
- **`sentinel()`** - Enables security checks and abuse protection

You can use either plugin independently, but using both together provides the full Reign Auth Infrastructure experience.
